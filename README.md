# NewLifeGame
## 製作期間
3時間クオリティ
## 使用言語
HTML/CSS, JavaScript
## ジャンル
思考ゲーム
## 概要
白の位置を操作します。
黒は1ターンに一つ増えます。
オセロのような要領で黒を囲んでいきます。
## 操作方法
移動元のセルをクリック、その後移動先のセルをクリック
## ゲームクリア
盤面を全て白にしたらクリアです。
## 書いた範囲
今のところ全て
## 制作動機
構想が降ってきたので。
一応設定的には世界を浄化していく的なストーリーがあったりなかったり。
## 頑張った所
キレイにマスを並べるところ。
## 面白いところ
段々白が増えていくところ。
